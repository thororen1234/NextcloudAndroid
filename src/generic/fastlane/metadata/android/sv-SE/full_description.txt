Självhostade produktivitetsplattformen som ger dig kontroll.

Funktioner:
* Enkelt, modernt gränssnitt, anpassat till temat på din server
* Ladda upp filer till din Nextcloud-server
* Dela dem med andra
* Håll dina favoritfiler och mappar synkade
* Sök i alla mappar på din server
* Automatisk uppladdning av foton och videor som skapas med din enhet
* Håll dig uppdaterad med notiser
* Stöd för flera konton
* Säkra åtkomst till din data med fingeravtryck eller PIN-kod
* Integration med DAVx5 (tidigare DAVdroid) för enkel synkning av kalendrar och kontakter

Rapportera alla problem på https://github.com/nextcloud/android/issues och diskutera appen på https://help.nextcloud.com/c/clients/android 

Ny till Nextcloud? Nextcloud är en privat server för filsynkronisering, delning och kommunikation. Det är fri programvara som du kan hosta själv eller betala ett företag för att göra det åt dig. På så sätt har du kontroll över dina foton, din kalender och dina kontakter, dina dokument och allt annat.

Kolla in Nextcloud på https://nextcloud.com
